# Known Fake and Lookalike Domains

Never link to any domain on this list. These are either outright fakes, parked lookalikes, or domains that used to be legitimate but have been abandoned or repurposed. When in doubt, verify against the real brand domain listed in `brands.md`.

## Confirmed fakes / lookalikes

| Fake domain | Real brand | Real domain |
|---|---|---|
| myevie.com | Evie Ring (Movano) | eviering.com |
| circul.com | Circular | circular.xyz |

| circular-ring.com | Circular | circular.xyz |
| circulsense.com | NOT Circular — sells an unrelated "Circul Ring" product that's easily confused with Circular Ring 2 | circular.xyz (if the user means Circular) |

## How to spot a lookalike

Common patterns that should make you suspicious of a domain:

- A prefix like "my," "get," "try," "the" in front of a brand name (my[brand].com, get[brand].com) when the real brand site is just [brand].com or [brand].ring
- A different TLD than the brand uses (e.g., brand.co when the real one is brand.com)
- A hyphenated version of the brand name (brand-ring.com instead of brandring.com)
- A subdomain on a completely unrelated domain (oura.someblog.com)

If a URL pattern-matches any of these, open the real brand's official site first (find it via their verified social media or a manufacturer lookup), and compare.

## When adding to this list

If you encounter a new lookalike in the wild while fact-checking, add it to the table above with a note on what it impersonates. Review this file periodically to keep it current.
